# Exam word groups for TELC B1-B2 text generation

Рабочая группировка слов для генерации учебных текстов. Цель — брать не случайные слова из частотного списка, а слова с высокой переносимостью: они должны помогать в письме, устной части, повседневных темах и типичных экзаменационных заданиях.

## 12. Conversational words: passive or informal only

Полезно понимать в речи, но в официальном письме TELC лучше заменять нейтральными словами.

- facet — парень / мужчина; лучше: mężczyzna
- koleś — чувак; лучше: osoba / znajomy
- kumpel — приятель; лучше: kolega / znajomy
- gość — гость / чувак; лучше: osoba / mężczyzna
- serio — серьёзно; лучше: naprawdę / rzeczywiście
- fajny — классный; лучше: interesujący / dobry / przyjemny
- gadać — болтать; лучше: rozmawiać
- wyluzuj — расслабься; лучше: uspokój się

## 13. avoid_for_exam_generator

Эти слова не стоит подмешивать в генератор экзаменационных текстов. Они могут пригодиться для пассивного понимания фильмов, но дают низкий экзаменационный возврат и часто портят стиль.

## Generator rule

При генерации одного текста брать:

1. одну тему из групп 5-10;
2. 6-10 тематических слов;
3. 3-5 связок из группы 2;
4. 2-4 слова мнения из группы 1;
5. не использовать слова из `avoid_for_exam_generator`.

Если есть сомнение, слово не удаляется из словаря. Его можно только исключить из генератора.
