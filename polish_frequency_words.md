# Частотные слова польского языка для TELC B1-B2

## Исследования и рекомендации

**Сколько слов нужно для уровня B1:**
- По исследованию Milton и Alexiou (2009): 2,750-3,250 слов
- Для уверенного разговорного B1: ~2,000-2,500 активных слов
- 2,500 самых частотных слов покрывают **92% разговорного польского**
- Для устной части экзамена TELC достаточно ~1,500 активных слов

**Принцип Ципфа:** 80% речи состоит из 20% самых частотных слов.
