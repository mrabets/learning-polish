# Evidence-based strategy for TELC B1-B2 vocabulary selection

Цель: сделать генерацию учебных текстов эффективнее для подготовки к TELC B1-B2 Polish, не распыляясь на случайные слова из кино/субтитров.

## Что показывает исследовательская база

1. **Сначала частотность и переносимость.** Высокочастотные слова дают лучший возврат на время обучения, но их нужно отбирать под цель: экзамен, повседневная коммуникация, письмо, говорение.
2. **Не одиночные слова, а продуктивные связки.** Для B1-B2 важны готовые фразы: мнение, причина, результат, контраст, пример, вывод. Они напрямую помогают писать письма/эссе и говорить связно.
3. **Группировка должна быть функциональной, а не алфавитной.** Для генерации текстов полезнее группы по экзаменационным функциям: здоровье, работа, технологии, путешествия, экология, отношения, аргументация.
4. **Низкопереносимый киношный шум нужно исключать из генератора.** Слова из субтитров часто частотны только из-за жанра: ругань, угрозы, оружие, полицейско-военные обращения, команды в драке. Они редко помогают на экзамене и могут ухудшить стиль письма.
5. **Удалять нужно консервативно.** Если слово может пригодиться в повседневной жизни или типичной теме B1-B2, оно остаётся. Убираются только слова, которые почти наверняка не стоит активно тренировать для экзаменационного текста.

## Практический алгоритм отбора

Каждое слово из `most popular polish words.txt` стоит относить к одной из четырёх зон:

### A. Must-use для генератора

Слова и фразы, которые повышают качество почти любого B1-B2 текста:

- логика: `mimo to`, `zamiast`, `oprócz`, `wobec`, `według`, `czyli`, `przecież`, `skoro`, `wcale`, `jedynie`
- мнение и оценка: `uważam, że`, `wydaje się`, `rzeczywiście`, `prawdopodobnie`, `obawiam się`, `żałuję`
- причина/результат: `powoduje`, `wpływ`, `skutki`, `prowadzić do`, `polega`, `zależeć`
- действия: `sprawdzić`, `zauważyć`, `przekonać`, `poradzić`, `osiągnąć`, `wymagać`

### B. Тематические слова для экзамена

Их нужно использовать в текстах по конкретным темам:

- здоровье/образ жизни
- работа/учёба
- технологии/медиа
- семья/отношения
- путешествия/город
- экология/общество
- личный опыт/истории

### C. Разговорные слова: оставить, но не использовать в официальном письме

Например: `facet`, `koleś`, `kumpel`, `gość`, `serio`, `fajny`, `gadać`, `wyluzuj`. Они полезны для понимания речи, но в TELC-письме лучше заменять на нейтральные варианты: `mężczyzna`, `osoba`, `znajomy`, `naprawdę`, `rozmawiać`.

### D. Stop-list для генератора экзаменационных текстов

Эти слова можно оставить только для пассивного понимания кино/сериалов. В учебные тексты для TELC их лучше не подмешивать.

Критерии попадания в stop-list:

- ругань/оскорбления;
- прямые угрозы и насилие;
- оружие, выстрелы, трупы;
- полицейско-военная и криминальная лексика, если она слишком сюжетная;
- слишком кинематографичные команды: «стреляй», «проваливай», «ложись» и т.п.

## Как генерировать новые тексты

1. Выбрать одну экзаменационную тему.
2. Взять 8-12 слов из группы A/B.
3. Добавить 3-5 связок для аргументации.
4. Сгенерировать текст на 120-180 слов.
5. Проверить, чтобы не было слов из stop-list.
6. После текста добавить 5-7 вопросов для говорения и 5 gap-fill предложений.

## Правило безопасности

Если есть сомнение, слово не удаляется из исходного словаря. Оно может быть перенесено в `avoid_for_exam_generator`, но не уничтожается полностью.
