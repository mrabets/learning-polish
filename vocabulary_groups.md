# Тематическая группировка слов для генерации текстов TELC B1-B2

## Научное обоснование

**Почему тематические группы эффективнее, чем случайный порядок:**

1. **Семантические сети (Nation, 2001; Schmitt, 2000):** слова, усвоенные в тематическом контексте, активируют семантическую сеть — мозг запоминает не слово, а узел в сети связанных понятий. Удержание лексики повышается на 30–40% по сравнению с изолированными списками.

2. **Принцип частотности (Zipf):** топ-2000 слов покрывают ~95% разговорного польского. Текущий файл уже отсортирован по частоте — это оптимально.

3. **Контекстуальное усвоение лучше карточек (Krashen, 1989):** встреча слова в тексте по знакомой теме в 3 раза эффективнее, чем флэшкарта. Поэтому текст должен быть «заточен» под группу слов, которые нужно отработать.

4. **Предупреждение о синонимах (Tinkham, 1997):** изучать синонимы/антонимы в одном уроке — хуже, чем разносить их по разным занятиям (интерференция). Тематические группы этого избегают.

5. **Экзаменационный washback-эффект:** если тренировочные тексты покрывают ровно те темы T01–T14, которые встречаются на TELC — слова активируются именно тогда, когда нужны.

---

## Как использовать при генерации текста

При генерации текста для темы TXX:
1. Включить в текст **как можно больше слов из группы TXX** ниже
2. Дополнить **универсальными словами** из `most popular polish words.txt` (все, что не в группах ниже)
3. Цель: покрыть все слова группы за 1–2 текста на эту тему
