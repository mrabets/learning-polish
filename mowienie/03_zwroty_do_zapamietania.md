# Каркас для заучивания — стандартные фразы (gotowiec)

> Это МИНИМУМ, который надо знать наизусть до автоматизма. Всё взято из «Setki» и официального документа Комиссии — именно так принято говорить на экзамене, без усложнений. Каждая тренировка начинается с повторения части этого файла.
>
> Статус заучивания отмечаем: `[ ]` не начато → `[~]` в работе → `[x]` автоматизм.

## A. Zadanie 1 — Opis fotografii: скелет из 12 фраз

Порядок ответа всегда один: **od ogółu do szczegółu** (общее → первый план → второй план → фон → погода/время → настроение).

- [x] 1. **Zdjęcie przedstawia…** *(вводное: кто/что + где — одним предложением)*
- [x] 2. **Na pierwszym planie widzimy…** *(главные люди/объекты)*
- [ ] 3. **Kobieta/Mężczyzna ma… lat, jest…** *(возраст «około 30 lat», рост, телосложение, волосы)*
- [x] 4. **Jest ubrany/ubrana w… / Ma na sobie…** *(одежда)*
- [~] 5. **Wygląda na… / Wydaje się…** *(впечатление: zmęczonego, studentkę, miłego człowieka)* — использовал `wyglądają na szczęśliwych`
- [x] 6. **Myślę, że… / Przypuszczam, że…** *(догадка: кем приходятся друг другу, что происходит)*
- [~] 7. **Osoby na zdjęciu + czasownik** *(действия: siedzi, stoi, rozmawia, bawi się, trzyma…)* — `całują się, uśmiechają się`
- [x] 8. **Na drugim planie… / Po lewej/prawej stronie… / W rogu…** *(второй план)*
- [x] 9. **W tle / W głębi widać…** *(фон)*
- [~] 10. **Jest słoneczny letni dzień. / Przypuszczam, że jest…** *(время года, суток, погода)* — путал `jest lato` / `latem`
- [x] 11. **Na zdjęciu dominują jasne/ciemne/ciepłe kolory.** *(цвета)*
- [x] 12. **Na zdjęciu panuje radosny/spokojny/wesoły nastrój.** *(настроение — финальная фраза)*
- [x] Доп.: **Ta sytuacja / To wydarzenie ma miejsce / odbywa się w…** *(универсальный оборот «где/когда происходит»)*
- [x] Доп.: **Moim zdaniem… / Uważam, że…** *(мнение в конце описания)*

Запасные: Na zdjęciu widzimy… • Prawdopodobnie… • Sądzę, że… • Na twarzy … maluje się uśmiech/smutek… • Przedmiot stoi/leży/wisi/znajduje się…

## B. Zadanie 2 — Monolog: универсальный скелет

**Wstęp (1–2 предложения):**
- [ ] Ostatnio wszyscy mówią o… / Popularnym tematem jest…
- [ ] Myślę, że warto… / Moim zdaniem… jest bardzo ważne.

**Rozwinięcie (3 пункта, по 2–3 предложения):**
- [ ] Po pierwsze… / Przede wszystkim…
- [ ] Po drugie… / Kolejną sprawą, o której chcę powiedzieć, jest…
- [ ] Po trzecie… / Muszę też powiedzieć, że…
- [ ] Na przykład… / Z mojego doświadczenia…
- [ ] Uważam, że… / Jestem przekonany/a, że…

**Zakończenie (1–2 предложения):**
- [ ] Podsumowując, … jest ważną częścią mojego życia / naszego życia.
- [ ] Na zakończenie mogę stwierdzić, że…

**Добавки по типу монолога:**
- *Рассказ о событии:* Pewnego dnia… → Najpierw…, potem… → Nagle… → W końcu… → Na szczęście wszystko dobrze się skończyło. → Ta przygoda nauczyła mnie, żeby…
- [ ] выучено
- *Фильм/книга:* Film, który ostatnio obejrzałem, to… → Akcja rozgrywa się w… → Główny bohater to… → Polecam ten film wszystkim. / Jestem pod wrażeniem tego filmu.
- [ ] выучено
- *Мнение:* Moim zdaniem… → Mój argument potwierdza fakt, że… → Z jednej strony…, z drugiej strony… → Bez wątpienia…
- [ ] выучено
- *Плюсы/минусы:* X ma zarówno zalety, jak i wady. → Co się tyczy zalet… → Warto jednak wziąć pod uwagę minusy… → Jednym z głównych argumentów przeciw jest fakt, że…
- [ ] выучено
- *Человек:* Poznałem ją/go w… / Znam ją/go od… → Ma … lat, jest… → Na co dzień ubiera się… → Jest życzliwa/pracowita/odpowiedzialna… → Zawsze mogę na niej/nim polegać.
- [ ] выучено

## C. Zadanie 3 — Dialog: обязательные реакции

**Официально (Pan/Pani — незнакомые, учреждения, телефон):**
- [ ] Dzień dobry, chciałbym/chciałabym + infinitiv (zarezerwować, zapytać, umówić się…)
- [ ] Czy może mi Pan/Pani powiedzieć…? / Chciałbym dowiedzieć się…
- [ ] Proszę mi powiedzieć, ile kosztuje… / Co jest wliczone w cenę?
- [ ] Czy jest możliwość…? / Czy mogę prosić o…?
- [ ] Dziękuję bardzo. Do widzenia. / Do usłyszenia.
- [ ] Przepraszam, czy nie przeszkadzam?
- [ ] Niestety, nie możemy przyjść, ponieważ… *(вежливый отказ)*
- [ ] Mam propozycję… / Może zrobimy tak: …? *(компромисс)*

**Неофициально (kolega/koleżanka):**
- [ ] Cześć! Co słychać? — Po staremu. / Wszystko w porządku.
- [ ] Słuchaj, mam pomysł… / Wiesz co… / Wyobraź sobie, że…
- [ ] Masz ochotę na…? / Może pójdziemy…? / A może pojedziesz ze mną?
- [ ] (+) Dlaczego nie! / No pewnie! / Z przyjemnością! (−) Chyba nie. / Nie mam ochoty. / Może innym razem.
- [ ] Nie martw się! / Zobaczysz, będzie super.
- [ ] Powinieneś/powinnaś… / Na twoim miejscu… *(совет)*
- [ ] Mówię ci, warto, bo… / Przecież… *(убеждение)*
- [ ] No dobrze, zgadzam się. / Jesteśmy umówieni? / To do zobaczenia w sobotę!

**Пожелания (składanie życzeń):**
- [ ] Wszystkiego najlepszego! / Moje gratulacje! / Zdrowia, szczęścia, miłości! / Powodzenia!

## D. Ratunkowe zwroty (когда не понял / забыл слово — ЭТО ТОЖЕ ОЦЕНИВАЕТСЯ КАК УМЕНИЕ)

- [ ] Przepraszam, czy może Pan/Pani powtórzyć?
- [ ] Nie zrozumiałem/zrozumiałam. Czy może Pan/Pani mówić wolniej?
- [ ] Co znaczy…? / Jak to się nazywa po polsku?
- [ ] Chodzi mi o to, że… *(перефразировать мысль)*
- [ ] Jak by to powiedzieć… / Chwileczkę… *(выиграть время вместо паузы)*
