# Официальные тесты egzaminu certyfikatowego B1 (dla dorosłych)

> Источник: **certyfikatpolski.pl** (Państwowa Komisja ds. Certyfikacji JPjO) — это официальные материалы самой Комиссии.
> Скачаны листы заданий (`arkusz`) и ключи/транскрипции (`klucz`/`transkrypcja`). Аудио для модуля аудирования не хранится в репозитории (чтобы не раздувать проект) — прямые ссылки ниже; агент скачает нужное аудио, когда дойдёшь до модуля «Rozumienie ze słuchu».
>
> **Как пользоваться:** распечатать `arkusz`, решить сам на время, потом разобрать ошибки с агентом. Ключ (`klucz`) — для самопроверки. У реальных тестов прошлых лет отдельного ключа нет, есть только транскрипция аудио.

Каждый тест = 5 модулей: Rozumienie ze słuchu, Rozumienie tekstów pisanych, Poprawność gramatyczna, Pisanie, Mówienie.

## Рекомендованный порядок
1. Сначала — **przykładowe** (есть ключи, удобно проверять себя).
2. Потом — **poprzednich_lat** как репетиции на время, ближе к октябрю 2026.
3. Модуль **Mówienie** тренируем отдельно с агентом (фото → монолог → диалог).

Всего: **15 тестов B1** (4 przykładowe + 11 poprzednich lat). Обновлено: сессии 2025 г. на сайте пока не выложены.
